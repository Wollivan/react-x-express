Mini React x Express Project

  • Create a React App
  • Create an Express Server
  • Have the React App call the Express server API
  • The React App requires:
    • Some sick CSS
    • Data from the API to be stored in state
    • Two seperate ways to trigger the API call to your Express Server
  • The Express Server requires:
    • A home route that gives some instructions on how to use the OTHER routes
    • Two other routes that return some sort of data
